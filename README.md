# NBA Performance SQL Analysis

## Overview
Analyzed NBA game performance data from 2004–2020 using SQL to identify trends
in scoring, three-point efficiency, and home-court advantage.

## Skills & Tools
- SQL (GROUP BY, HAVING, ORDER BY, aggregation)
- Time-series trend analysis
- Sports analytics and performance evaluation

## Key Insights
- Identified long-term scoring and three-point shooting trends
- Evaluated the relationship between shooting efficiency and team success
- Analyzed changes in home-court advantage over time

## Repository Contents
- `queries.sql` – SQL queries used for analysis
- `milestone.pdf` – Completed project submission with results and explanations
